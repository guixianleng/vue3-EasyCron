# easy-cron

easy-cron 是基于 `Vue3.js` 和 `element-plus` 实现 crontab 表达式的 UI 组件，支持多种特性，可以方便、快捷直观地定义 cron 表达式，并支持给定开始时间后的执行时间点，方便排除错误。
